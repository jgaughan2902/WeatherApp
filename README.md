# Welcome to my Weather App!

I've created this experimental weather app which pulls data directly from the National Weather Service public API.

Provided in this repository is the single .py file used to create the current version of the app. Additionally, this README and the package requirements file are included.

The intention is to update it in the future to add functionality.

### Version 1.0.1

Code updated to include caching mechanisms to (hopefully) improve performance.

### Version 1.0.0

Initial code imported into repository for deployment.
